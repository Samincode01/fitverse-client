# FitVerse

**Live Site:** https://fitverse-client.vercel.app

## Project Overview

FitVerse is a modern full-stack fitness platform designed to connect users with professional trainers through interactive fitness classes. The platform features a secure authentication system, role-based dashboards, trainer applications, favourite classes, and comprehensive admin management tools.

The application supports three different roles:

* **User** – Browse and save classes, apply as a trainer, and manage personal activities.
* **Trainer** – Create and manage classes, interact through forums, and monitor enrolled students.
* **Admin** – Manage users, trainers, classes, trainer applications, forum posts, and platform statistics.

---

## Key Features

### Authentication & Authorization

* Email and Password Authentication
* Google Sign In
* Secure authentication using Better Auth
* Role-based private routes
* Persistent user sessions

### User Features

* Browse all fitness classes
* View detailed class information
* Add classes to favourites
* Apply to become a trainer
* View trainer application status and admin feedback
* Personalized dashboard with statistics

### Trainer Features

* Trainer dashboard with analytics
* Add and manage classes
* View enrolled students
* Add forum posts
* Manage personal forum posts

### Admin Features

* Dashboard with platform statistics
* Manage users
* Block and unblock users
* Approve or reject trainer applications
* Manage trainers
* Manage classes
* Moderate forum posts
* View transactions

### Additional Features

* Responsive design for all devices
* Image upload with ImgBB
* Toast notifications
* Confirmation dialogs with SweetAlert2
* Modern UI with Tailwind CSS
* Dynamic role-based dashboard layout
